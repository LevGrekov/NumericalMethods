Этот проект создан для решения задач по численным методам, связанным с построением линейных и кубических сплайнов. Программа также умеет строить графики для визуализации результатов интерполяции.

## Возможности программы

1. **Линейные сплайны:** Программа способна вычислять линейные сплайны для набора точек.
2. **Кубические сплайны:** Реализован расчет кубических сплайнов для более гладкой интерполяции.
3. **Полиномы Лагранжа в форме Ньютона:** Программа вычисляет полиномы Лагранжа в форме Ньютона для предоставленных данных.
4. **Графики:** Позволяет визуализировать построенные линейные и кубические сплайны, а также полиномы Лагранжа.
